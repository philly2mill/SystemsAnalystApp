

A Progressive Web App designed to help users practice and simulate key tasks for a Systems Analyst role, especially focused on healthcare IT environments.  
Build with React, React Router, and designed as a PWA for installability on desktop and mobile devices.



## Features

- Multi-page app simulating real-world Systems Analyst tasks  
- User input and interactive practice modules (Requirements gathering, Support triage, Project management, Documentation)  
- Responsive UI with a clean, focused design and hamburger menu  
- Embedded study music playlist area for focus enhancement  
- Offline-capable via PWA service worker  
- Installable on supported browsers/devices  



## Technologies Used

- React  
- React Router DOM  
- Vite (build tool and dev server)  
- CSS for styling  
- PWA manifest and service worker  



## Getting Started

### Prerequisites

- Node.js (v16 or newer recommended)  
- npm (comes with Node.js)
